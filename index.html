    Projects

    Interactive A/B Test Explorer

    Built a self-serve A/B testing tool using Streamlit. Supports statistical testing, XGBoost-based conversion prediction, and automated PowerPoint report generation. Designed for analysts to explore results and share insights with minimal effort.

    FutureAQI Forecast – Interactive Air Quality Forecasting App

    Air Quality Forecasting App – An interactive web application built with Streamlit to fetch, visualize, and forecast Indian air quality data using Prophet. Includes real-time API integration, EDA visualizations, and 90-day AQI predictions for major cities.

    FutureAQI Forecast – Automated Forecasting Pipeline

    Air Quality Forecasting Pipeline – A modular Python script-based workflow to collect, clean, analyze, and forecast AQI data using the OpenAQ API and Prophet. Includes data logging, forecasting evaluation, and automated processing across locations.

    RAG-Based LLM Assistant

    Built a document-aware chatbot using Retrieval-Augmented Generation (RAG). Combined semantic search (FAISS + Sentence Transformers) with a lightweight LLM (Phi-1.5) to answer questions grounded in uploaded PDFs, with fallback to model knowledge when context is weak.

    Fairness Evaluation of AI Models

    Developed a toolkit to assess fairness in machine learning models using metrics like demographic parity and equal opportunity. Included visualizations and mitigation strategies to detect and reduce bias across sensitive attributes such as gender and race.
